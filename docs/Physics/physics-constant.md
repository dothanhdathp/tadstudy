# \[Vật Lý\] Hẳng Số

## 

| Tên                                                  |     Ký Hiệu      |             Giá Trị             |               Đơn Vị SI                |
| :--------------------------------------------------- | :--------------: | :-----------------------------: | :------------------------------------: |
| __Tốc độ ánh sáng trong chân không__                 |       $c$        |    $\approx 299,792,458...$     |              $\text{m/s}$              |
| __Hằng số Planck__                                   | $\hbar$ hoặc $h$ | $\approx 1.381 \times 10^{-23}$ |        $\text{J}\cdot\text{s}$         |
| __Hằng số Boltzmann__                                | $k$ (hoặc $k_B$) | $\approx 1.381 \times 10^{-23}$ |             $\text{ J/K}$              |
| __Độ lớn điện tích cơ bản (Điện tích của electron)__ |       $e$        | $\approx 1.602 \times 10^{-19}$ |               $\text{C}$               |
| __Hằng số hấp dẫn__                                  |       $G$        | $\approx 6.674 \times 10^{-11}$ | $\text{ N}\cdot\text{m}^2/\text{kg}^2$ |
| __Số Avogadro__                                      | $N_A$ (hoặc $L$) | $\approx 6.022 \times 10^{23}$  |           $\text{ mol}^{-1}$           |
| __Hằng số khí lý tưởng__                             |       $R$        |       $\approx 8,314...$        |     $\text{ J/(mol}\cdot\text{K)}$     |
