# \[Tools\] GNU Octave

Công cụ dùng để __vẽ đồ thị__ trên hệ điều hành nhân Linux.

## Tải về

=== "Debian/Ubuntu"
    ```bash
    sudo apt-get install octave
    ```
=== "Fedora/RHEL"
    ```bash
    sudo dnf install octave
    ```
    Để cài phiên bản cũ hơn:
    ```bash
    sudo yum install octave
    ```
=== "macOS"
    __Using Homebrew:__
    ```bash
    brew install octave
    ```
    __Using MacPorts:__
    ```bash
    sudo port install octave
    ```
=== "Windows"
    Vào trang web [GNU Octave website's download](https://octave.org/download)
    Tải về tệp: __octave-10.3.0-w64-installer.exe__
